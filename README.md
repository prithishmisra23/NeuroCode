# NeuroCode

AI-powered code intelligence extension for VS Code.

## Features

- **Repository Scanning**: Deep analysis of project structure and dependencies.
- **Architecture Visualization**: Interactive dependency graphs.
- **Bug Prediction**: Risk scoring based on complexity and history.
- **AI Refactoring**: Intelligent code improvements with diff previews.
- **Developer Dashboard**: Comprehensive overview of project health.

## Installation

1. Clone the repository.
2. Run `npm install`.
3. Press `F5` in VS Code to start debugging.
